# VoiceCode Demo Project

This is a sample TypeScript/React project created for demonstrating VoiceCode extension features in screenshots.

## Features

- **React Components**: Button component with TypeScript
- **Utility Functions**: Helper functions for common tasks
- **TypeScript**: Full type safety
- **Clean Code**: Well-structured and readable

## Usage

This project is designed to showcase VoiceCode's capabilities:

1. **Voice-to-Code**: Generate components using voice commands
2. **AI Assistance**: Get code explanations and refactoring suggestions
3. **Multi-File Editing**: Make coordinated changes across files
4. **Code Navigation**: Jump between files using voice

## Voice Command Examples

Try these voice commands with VoiceCode:

- "Create a new React component called UserProfile"
- "Add a function to format phone numbers"
- "Refactor the Button component to use hooks"
- "Explain what the debounce function does"
- "Add error handling to the formatDate function"

## Getting Started

1. Open this project in VSCode
2. Install VoiceCode extension
3. Press `Ctrl+Shift+V` to start voice coding
4. Try the voice commands above!

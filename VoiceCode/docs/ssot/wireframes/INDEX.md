## Wireframe Format

When created, wireframes should be:

- Text-based ASCII wireframes in markdown files: `docs/ssot/wireframes/<SCR-ID>.md`
- Or linked to external Figma/design tool URLs
- Each wireframe file must reference: Screen ID, Platform, FLOW, Components used, Data displayed

## 📋 Quick Reference

### VSCode Shortcuts
- **Start Listening**: `Ctrl+Shift+V`
- **Show Chatbox**: `Ctrl+Shift+C`
- **Command Palette**: `Ctrl+Shift+P`
- **Settings**: `Ctrl+,`
- **Chat Panel**: `Ctrl+Shift+I`

### Recommended VSCode Settings
- **Theme**: Dark+ (default dark) or GitHub Dark
- **Font**: Fira Code or JetBrains Mono
- **Font Size**: 14-16px
- **Zoom**: 100% (Ctrl+0)
